# Fun117 について

## 概要

日本を拠点に活動している情熱的なプログラマーです。特にNext.jsやWeb開発に専念しています。Webアプリケーション、ウェブサイト、eコマースプラットフォーム、データベースの開発に積極的に関わっています。最近は CLI やフレームワークの開発などを始めました。


## 貢献方法

コミュニティからの貢献を歓迎します！以下の方法で貢献できます：

### 問題の報告

ウェブサイトで問題が発生した場合は、[GitHub Issues](https://github.com/fun117/fun117/issues) ページに報告してください。問題を報告する際には、可能な限り詳細な情報（再現手順や関連するエラーメッセージなど）を含めてください。

### プルリクエストの送信

バグ修正、新機能、改善のためのプルリクエストを受け付けています。プルリクエストを送信するには：

1. GitHubでリポジトリをフォークします。
2. フォークしたリポジトリをローカルマシンにクローンします。
3. 変更用の新しいブランチを作成します。
4. 変更を加え、明確なコミットメッセージとともにコミットします。
5. フォークしたリポジトリに変更をプッシュします。
6. 元のリポジトリにプルリクエストを作成し、変更内容を説明します。
